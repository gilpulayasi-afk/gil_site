Certificate Program in Advanced Web Technology

The objective of this program is to provide students with expertise in Website development using HTML, CSS, Javascript, PHP and SQL.
Duration : 16 - 18 Weeks (320 Hours)     
Eligibility : Minimum of Grade 12 Certificate    
Program Prerequisite :Sound knowledge of Computing Fundamentals.

SL No. Courses Hours
1 Fundamentals of Computer and OS Concepts 40
2 Client Side Web Technologies 40
3 Database Concepts 30
4 Server Side Web Technologies (PHP ,Laravel) 100
5 Internet Terminologies and Application Deployment 30
6 Management Development Program 40
7 Capstone Project 60

Outcome : The Certificate Program in Advanced Web Technology (CCAWT) aims to groom the students to enable them to work on current web technology scenarios, as well as to prepare them to keep pace with the changing face of technology and the requirements of the growing IT industry. After the completion of a program, students can work as Web Developers / Web Designers / IT Support Staff.
