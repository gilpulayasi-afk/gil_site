Certificate Program in Java Web Development

The objective of this program is to provide students with expertise in Java Web Development. This includes both the Core Java and Advanced Java programming. After completing the program, the students will be able to design, develop, and maintain web-based enterprise applications effectively.
Duration : 16 - 18 Weeks (320 Hours)
Eligibility : Minimum of Grade 12 Certificate  
Program Prerequisite : Fair knowledge of Computing Fundamentals

SL No. Courses Hours
1 Fundamentals of Computer and OS Concepts 20
2 Software Development Life Cycle 10
3 Database Concepts 30
4 Client Side Web Technologies 40
5 Java Programming 50
6 Enterprise Java 70
7 Management Development Program 40
8 Capstone Project 60
