Certificate Course in Big Data Technologies

The objective of this course is to provide the student with an expertise in Big Data Technologies. This includes concepts and use of big data technologies along with concepts of Linux programming, Python programming, Machine Learning. After doing the course, the student will be able to use and provide various solutions using Big Data Technologies.
