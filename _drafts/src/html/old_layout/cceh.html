Certificate Course in Ethical Hacking & Information Security

This course is aimed to provide skills on security programming which will help the students who want to make a career in security domain.
Duration : 320 Hrs      Course Fee : K----
Eligibility : Minimum of Grade 12 Certificate     NSQF Level : 7
Course Prerequisite : Candidate should have knowledge of computer & networking fundamentals and Basic Computer Programming with OOPs concepts.

SL No. Modules Hours
1 Java Programming with Crypto API 80
2 Application Security 70
3 Ethical Hacking 80
4 Management Development Program 40
5 Project 50

Outcome : This course is aimed to provide skills on security programming which will help the students who want to make a career in security domain as well as prepare them to keep pace with the changing face of technology and the requirements of the growing IT industry.
These candidates will be trained in Application Security, Ethical Hacking and Management skills. After the completion of the course, students can work in the area of Application Security or Ethical Hacking.
